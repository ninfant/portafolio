Feature-Flag Microservice

Built a secure feature flag microservice that allows dynamic enabling/disabling of features by customer and region. Includes a protected REST API, an admin dashboard with JWT based authentication, and a consumer microservice that reacts to live flag changes in real time enabling safe, controlled releases without redeployments or service interruption.

Developed a reusable Redux-based state management system with async thunks to manage flags, authentication, and API calls efficiently. Created a modular backend using Express, PostgreSQL, and Knex.js, supporting scoped feature flags by customer and region with atomic flag toggling. Designed a live consumer demo microservice that conditionally executes logic based on real-time flag state, showcasing production-readiness and developer flexibility.
